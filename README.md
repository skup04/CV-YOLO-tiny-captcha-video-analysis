# CV-YOLO-tiny-captcha-video-analysis

## Описание

Проект демонстрирует анализ изображений и видео с использованием модели **YOLOv4-Tiny** для задачи детекции объектов.  
Используется облегчённая версия YOLO, которая обеспечивает высокую скорость работы при приемлемой точности.

Проект реализован в виде Jupyter Notebook и подходит для учебных и исследовательских целей в области компьютерного зрения.

## Возможности

- Детекция объектов на изображениях  
- Анализ видео по кадрам  
- Использование предобученной модели YOLOv4-Tiny  
- Визуализация bounding box и подписей классов  
- Работа в реальном времени (при наличии камеры)

## Установка

Клонирование репозитория:

```bash
git clone https://github.com/skup04/CV-YOLO-tiny-captcha-video-analysis.git
cd CV-YOLO-tiny-captcha-video-analysis
```

Установка зависимостей:

```bash
pip install opencv-python notebook
```

---

## Запуск

Запустите ноутбук:

```bash
jupyter notebook Photo-Video_CV.ipynb
```

Внутри ноутбука:
- загрузите изображение или видео  
- инициализируйте модель  
- выполните детекцию
